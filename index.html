GitHub token
Kept in this tab's sessionStorage only — sent nowhere except api.github.com.
Without a token GitHub allows only 60 API requests/hour, so the dashboard runs in lite mode (releases and open PRs only). A token unlocks 5,000/hour plus the main-vs-release, test-bed and per-service columns.

Public repos: any PAT, no scopes needed. The test-bed column reads the private archive-test-bed repo: classic PAT with repo, or a fine-grained token with read access to the ghga-de repositories. Create one.
